# ⚡ Auto Add Friend Facebook (Chrome Extension)

A lightweight Chrome Extension to automate sending friend requests on Facebook with smart delay, mutual friends filtering, and human-like behavior.

---

## 🚀 Features

- ⏱ Custom delay (min & max interval)
- 👥 Mutual friends filter
- 🔁 Auto scroll & continuous detection
- 📊 Real-time progress tracking
- 🛑 Start / Stop control
- 🧠 Human-like interaction (safer usage)

---

## 📦 Installation (Manual)

Since this extension is not published on Chrome Web Store, you need to install it manually:

### 1. Download / Clone Repository

```bash
git clone https://github.com/okthapian/crome_extention_add_friend_fb.git
```

Or download ZIP and extract it.

---

### 2. Open Chrome Extensions

Go to:

chrome://extensions/

---

### 3. Enable Developer Mode

Turn on **Developer mode** (top right corner)

---

### 4. Load Extension

Click:

Load unpacked

Then select the folder:

crome_extention_add_friend_fb

---

## ▶️ How to Use

1. Open Facebook (recommended page below 👇)
2. Click the extension icon
3. Set your preferences:
   - **Interval (seconds)** → delay between clicks
   - **Max Add Friend** → limit per session
   - **Only mutual friends** → filter users
4. Click:
   ▶️ Start Adding
5. Monitor progress in the popup
6. Click:
   ⛔ Stop  
   to stop anytime

---

## 🌐 Recommended Page

Use this page to find "Add Friend" buttons:

👉 https://web.facebook.com/friends/suggestions

This page shows **People You May Know**, which is ideal for this extension.

## ⚠️ Disclaimer

- This tool is for educational purposes only.
- Use at your own risk.
- Excessive automation may lead to temporary restrictions from Facebook.

Recommended:
- Use delay ≥ 60 seconds
- Limit requests per day

---

## 🧠 Tips for Safe Usage

- Use **mutual friends filter**
- Avoid adding too many people in a short time
- Act like a normal user
- Do not run continuously for hours
